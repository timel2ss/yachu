## ✔ 요구사항

1. 주사위는 1부터 6까지의 값을 가질 수 있다.
2. 주사위를 굴린다.
3. 주사위를 고정시켜 리롤할 주사위를 고를 수 있다.
4. 주사위 조합에 따라서 족보를 맞출 수 있다.
5. 족보에 맞는 점수를 획득하여 저장할 수 있다.
